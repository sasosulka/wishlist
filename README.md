### üá∫üá∏ Project Description
A stylish wish list application built with a **Neobrutalism** aesthetic. I developed this project to manage personal gift ideas and easily share them with friends.

**Key Features:**
* Add wishes with prices and shop links.
* Upload images from your device or use web links.
* Automatic total cost calculation.
* "Share" button: generates a text-based list to send via messaging apps.
* Data persistence using LocalStorage.
